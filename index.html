Project idea

Tech-aid.com.au

Overview

“Tech-aid.com.au” is a website aiming to provide solutions for novice and average computer users who are experiencing technical issues. The website contains key features for technical assistance and troubleshooting such as: an FAQ, a list of common issues, guides for software/hardware instalment and updates, live chat, demonstration videos, and remote access. Tech-aid provides clear instructions and uses language that is comprehensible by users without technical experience. The goal of Tech-aid is to provide easily accessible information and help to individuals who are not able to solve their own problems.

Motivation

This project will be useful for users of all ages, abilities, and knowledge regarding computer issues. Older adults, for example, often feel outpaced by the vast amount of information and learning that coincides with the increase in the use of technology. When they encounter a problem, they might not know where to turn for help, or even realise that they need assistance in the first place. Tech-aid will offer troubleshooting and other forms of help in an easily digestible method, attempting to not only fix the issue at hand but also teach the user about basic computer functions and utilisations.

Description

Tech-aid will run online as a HTML and CSS based website. Tech-aid will also have elements of JavaScript to utilise Live Chat. The website will feature large, clickable hyperlinks as headings that will redirect the user to the correct page or section. The hyperlink headings will take the user to: an FAQ, a list of common issues, guides for software/hardware instalment and updates, a live chat feature, and remote access setup. There will also be a glossary section in case the user is unsure about what a certain word or term means.

CSS will be utilised to give the UX of the website a dark grey background and making the text a bright white colour. All font size will be slightly larger than the average website, have a line length limit of 80 characters and include a line spacing of 150%. This CSS styling is designed with the visually impaired in mind to provide the most accessible and user-friendly experience.

The FAQ will list general questions as hyperlinks, with a search box near the top of the page where the user can type in keywords to highlight a specific question. Some questions will require a long or in-depth explanation, in which case a separate tab will open with the relevant information, context, examples, and answer. These questions will include “What is hardware?”, “Why does my computer restart without warning?”, “How do I clean my keyboard?”, and “What do I do when I’ve forgotten my password?”, as examples.

A list of common issues will also be available, attempting to troubleshoot and guide the user through their problem. Like the FAQ, there will also be a search feature near the top of the page so the user can search for keywords and quickly find their issue. Common issues that everyday users can face include, “no internet/Wi-Fi connection”, “Windows running slow or freezing”, “process and application crashes or not responding”, and “freeing up hard drive space”. By clicking on any of these links, a separate tab will open with the relevant instructions and context for the issue.

There will be a section of the website devoted to software and hardware instalment/upgrades. By clicking on this section titled “Guides”, the user will be directed to a page with different topics about installing both software and hardware; and upgrading them if they are outdated. Each topic will include relevant videos showing a screen recording or actual video of the IT support individual installing/upgrading the software/hardware. These will be YouTube videos embedded in the topic page, so they are easily viewable by the user. As an example, there will be a video of the IT support individual opening their computer case, removing a hard drive, and replacing it. The steps taken in the video will be written in a step-by-step text format on the Tech-aid Guides page, and will have both audio commentary and Closed Captioning features on the video itself

The live chat feature will be implemented by “Crisp” which is a free multichannel messaging platform. It will allow the user to ask any further questions inside a chat box in the bottom right-hand corner of the screen. Crisp will send a notification to the IT support individual as soon as a message is sent, allowing for an instant response or feedback to the user. This feature will be useful as a technical issue or query may not be covered on the website, but still be solvable by the IT support individual.

Remote access may be needed or requested by some users as their technical issues might not be covered on Tech-aid. This gives the IT support individual control over the user’s computer without needing to be located physically at their computer. Remote access can be utilised to gain information about the technical issue, or even allow the IT support individual to fix the problem entirely. This will all be done through TeamViewer and there will be a step-by-step guide located on the “Remote Access” page showing the user how to install and set up the program.

Tools and Technologies

Software needed to create and manage the website includes: “Atom” to build the webpage in HTML & CSS, “Crisp” to use the features of live chat, and TeamViewer to handle the remote access and screen capture. The Tech-aid website will be hosted by https://au.godaddy.com/ which provides unlimited websites, bandwidth and 24/7 security monitoring.

Skills Required

  • HTML and CSS proficiency to create an appealing and easily accessible website which users with vision impairments can understand
  • Troubleshooting skills
  • Communication and patience with users
  • Use of special programs/applications such as "Crisp", "TeamViewer", and "Atom".
  • Installing and updating software and hardware

All the skills listed above are feasible to attain with practice and focus. Crisp and TeamViewer are both paid software which are designed to be easily understandable by both the user and the IT support individual. Hardware instalment is straight forward and teachable to the novice computer user if they follow the guide made for them. The troubleshooting skills needed may require some trial and error to learn, however, with the amount of information available on the internet most issues that will arise will have an answer or fix already.

Outcome

If the project is successful, valuable support and knowledge will be given to novice or impaired computer users. Their immediate issues will be solved with the help of an IT support individual, and they will gain some technical skills of their own in the event of a future problem. The website has been designed with the novice user in mind to be easily accessible and understandable by individuals of any skill level. Tech-aid will also give the user the confidence to reach out and ask for support once they see how simple it can be to get effective troubleshooting assistance.
